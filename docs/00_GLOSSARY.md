# docs/00_GLOSSARY.md — Глоссарий PROCLUSTER

> Единый словарь терминов. Цель: MiMo одинаково понимает термины во всех сессиях.
> При сомнении в значении термина — сверяйся с этим файлом.

---

## Рынок и данные

- **Тик (tick)** — минимальный шаг цены инструмента (tickSize). Futures BTCUSDT = 0.1$, Spot = 0.01$.
- **Трейд (trade)** — отдельная сделка с биржи (цена, объём, время, tradeId, isBuyerMaker).
- **trades stream** — поток тиковых сделок Binance (используем его, не aggTrade).
- **isBuyerMaker** — флаг Binance. true → агрессор продавец → BID; false → агрессор покупатель → ASK.
- **tradeId** — порядковый ID сделки. По нему сортируем и контролируем непрерывность.
- **trade time (T)** — время сделки. Используем именно его для привязки к свече (не event time).

## Объёмы (market-side)

- **Bid** — объём РЫНОЧНЫХ ПРОДАЖ (продажа по рынку в бид). isBuyerMaker=true.
- **Ask** — объём РЫНОЧНЫХ ПОКУПОК (покупка по рынку в аск). isBuyerMaker=false.
- **Volume** — суммарный объём на уровне = bid + ask.
- **Delta** — bid/ask дисбаланс свечи = Σ(ask − bid). Положительная = перевес покупок.
- **CVD (Cumulative Volume Delta)** — накопительная сумма дельт свечей за диапазон.

## Кластеры и сжатие

- **Кластер / ячейка (ClusterCell)** — диапазон цен в свече с агрегированными bid/ask/volume.
- **Сжатие (compression)** — сколько тиков объединяется в один кластер.
  Futures база 25 (=2.5$), Spot база 500 (=5$). До 10 уровней, кратных базе.
- **binSize** — размер кластера в $ = compression × tickSize.
- **Бин (bin)** — конкретный ценовой диапазон [binPriceLow, binPriceHigh).
  Ключ кластера = binPriceLow (нижняя граница).
- **Базовый уровень (×1)** — единственный, что хранится в ClickHouse.
  Высшие уровни (×2…×10) — объединение базовых на лету или из Redis-кэша.

## Виды свечей

- **Кластера (Clusters)** — footprint с числами bid/ask по каждому ценовому уровню. Близкий зум.
- **Футпринт (Footprint)** — кластерный вид + гистограмма объёма. Средний зум.
- **Японские свечи (Candlesticks)** — классические OHLC свечи. Дальний зум.
- **Бары (Bars)** — OHLC барами.
- **Auto** — авто-переключение по числу видимых свечей: <70 кластера, 70–300 футпринт, ≥300 японские.

## Аналитика свечи

- **POC (Point of Control)** — ценовой уровень с максимальным горизонтальным объёмом в свече.
- **Value Area (VAH / VAL)** — диапазон вокруг POC, вмещающий 70% объёма свечи.
  VAH = верхняя граница, VAL = нижняя.
- **Имбаланс (диагональный, ATAS-стиль)** — перевес одной стороны над соседним уровнем:
  bid imbalance: ask[price] > bid[price−1] × ratio; sell imbalance: bid[price] > ask[price+1] × ratio.
  Порог по умолчанию >300% (ratio = 3). Сравнение по соседнему БИНУ, не тику.
- **bid/ask skew (перевес)** — % доминирования стороны в окне (Cluster Search). Формула в docs/03.

## Стакан (DOM)

- **DOM (Depth of Market)** — стакан: лимитные bid/ask заявки по уровням цены.
- **Снапшот стакана** — мгновенное состояние DOM в момент закрытия интервала.
  Futures раз в 1м, Spot раз в 15м. На старте history НЕ пишем (только структура).
- **Шаг плотности** — агрегация стакана по base_compression (Futures 2.5$, Spot 5$).
- **Heatmap (тепловая карта)** — будущая фича лимитных плотностей. СЕЙЧАС НЕ делаем.

## Индикаторы

- **Cluster Search** — базовый индикатор поиска крупных объёмов (логика ATAS). Детали docs/03.
- **Volume on Chart** — гистограмма объёма поверх графика (серый/зелёный/красный по delta).
- **Пресет (preset)** — сохранённый набор настроек индикатора. Дефолтный пресет меняют только VIP/Admin.

## Инфраструктура

- **ClickHouse** — БД для market data (кластеры, снапшоты). Жёсткий TTL логов.
- **SQLite** — БД пользователей, настроек, тарифов, подписок, пресетов.
- **Redis** — live-агрегация текущей свечи + кэш последних ~700 свечей + кэш высших сжатий.
- **WS-хаб** — раздаёт realtime-апдейты кластеров на фронт (батч 100–500ms).
- **Caddy** — reverse-proxy: TLS, HTTPS, rate-limit для обоих доменов.

## Тарифы

- **Guest / Free / Pro / VIP / Admin** — роли. Детали и лимиты в docs/04_TIERS.md.
- **history_days** — глубина истории по роли (Guest 7, Free 180, Pro настр., VIP вся).

## Домены

- **procluster.online** — лендинг (презентация возможностей).
- **chart.procluster.online** — терминал (графики, стакан, индикаторы).
